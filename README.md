# MATLAB Image Processing Application

## Project Overview

This project was developed as part of the Machine Sensing module using MATLAB.

The application implements multiple image smoothing and enhancement techniques to reduce noise in digital images while preserving image quality.

A GUI-based image processing application was developed using MATLAB App Designer to allow users to apply filters interactively and compare results visually.

## Filters Implemented

### Conservative Smoothing Filter
- Adaptive noise reduction technique
- Preserves image edges
- Replaces only extreme pixel values

### Median Smoothing Filter
- Effective for salt-and-pepper noise removal
- Maintains edge sharpness
- Non-linear filtering method

### Mean Smoothing Filter
- Linear averaging filter
- Reduces Gaussian noise
- Smoothens image intensity variations

## System Workflow

1. User uploads an image into the application
2. Selected filter or enhancement technique is applied
3. MATLAB processes the image using implemented algorithms
4. Processed output image is displayed in the GUI
5. User can save the final enhanced image
